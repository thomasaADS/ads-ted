# 🚀 מדריך התקנה - Boosti AI Campaign Generator

## 📋 מה חדש?

האתר שודרג עם:
- ✨ **סוכן AI חכם** - ממשק שיחה אינטראקטיבי ששואל שאלות ומוביל אותך ליצירת קמפיין
- 🤖 **Gemini AI Integration** - יצירת תוכן קריאטיבי ומקצועי עם AI של Google (חינמי!)
- 🎨 **עיצוב מודרני** - landing page בסטייל מקצועי עם אנימציות חלקות
- 🎯 **חוויית משתמש משופרת** - צעד אחר צעד עם פידבק ויזואלי

---

## ⚙️ הגדרת Gemini AI (חינמי)

### 1. קבל API Key מ-Google

1. לך ל: **https://makersuite.google.com/app/apikey**
2. התחבר עם חשבון Google שלך
3. לחץ על **"Create API Key"**
4. העתק את ה-API Key

### 2. הגדר את המפתח בפרויקט

1. צור קובץ בשם `.env` בתיקיית הפרויקט (ליד `package.json`)
2. הדבק את זה:

```env
VITE_GEMINI_API_KEY=YOUR_API_KEY_HERE
```

**החלף את `YOUR_API_KEY_HERE` במפתח שקיבלת!**

### 3. התחל את השרת

```bash
npm run dev
```

**זהו! עכשיו ה-AI יעבוד! 🎉**

## 🛠️ פתרון בעיות

### AI לא עובד?
1. ✅ בדוק שה-API key נמצא ב-`.env`
2. ✅ הפעל מחדש את השרת (`npm run dev`)
3. ✅ בדוק שיש לך אינטרנט
4. ✅ בדוק ב-Console אם יש שגיאות

אם עדיין לא עובד - המערכת תעבור ל-Fallback mode ותשתמש בתבניות מוכנות.

### שגיאות בהרצה?
```bash
# נקה את node_modules והתקן מחדש
rm -rf node_modules
npm install
npm run dev
```
